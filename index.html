To Do List App

  • purpose: An app that's designed to track completed and uncompleted tasks with set deadlines
  • tech: react native, nodeJS, Expo
  • skills: expo for cross-platforming, react native UI components, real time updates with state management
  • problem: separating the completed and uncompleted tasks from each other, I had to completely switch methods and just filter out the comleted and uncompleted tasks it worked. The best solutions are simplest ones, and that's the biggest thing I learned in this project: always go for the simplest answer.
  • highlight: The color scheme used is designed to motivate (red) and focus (blue) the user. The overall look of the app is to be reminiscent of a corkboard set up, moving posted notes fromone category to the other to keep track of the days accomplishments.
